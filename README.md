# User Management API

This project is part of the **Peer‑graded Assignment: Building a Simple API with Copilot**.  
It demonstrates how to build and enhance a simple ASP.NET Core API with CRUD operations, validation, middleware, and Swagger documentation.

## ✨ Features
- CRUD endpoints: GET, POST, PUT, DELETE
- Validation for user input (required fields, email format)
- Middleware for request logging and exception handling
- Swagger/OpenAPI documentation with XML comments
